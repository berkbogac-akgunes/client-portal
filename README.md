# Client Portal (React)

Authentication-focused React application integrated with a Spring Boot backend.

## Tech Stack
- React
- React Router
- Context API
- Axios
- Tailwind CSS

## Features
- User registration & login
- Protected routes
- Persistent auth state (localStorage)
- Backend API integration
