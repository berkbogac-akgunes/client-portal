# Client Portal (React)

  Authentication-focused React application integrated with a Spring Boot backend.

  This frontend demonstrates secure authentication flow, protected routing, and backend API integration using a clean state management approach.

## Tech Stack

  React

  React Router

  Context API

  Axios

  Tailwind CSS

## Features

  User registration & login flow

  Protected routes using custom route guards

  Persistent authentication state (localStorage)

  Backend API integration with Axios

  Basic error handling for authentication failures

## Authentication Flow

  User submits credentials

  Backend validates and responds

  Auth state stored via Context API

  Protected routes accessible only when authenticated

  Auth state persisted across page refresh
